# Password-Manager-v1.0

>A simple Password Manager built with Python

**Struggling with passwords security? Worried about losing handwritten note?**

This program is a lightweight ultility designed for multi-user password management, aimed at providing a robust solution for storing and managing passwords offline.

## Functions
- User Registration
- User Data Isolation
- Input Validation
- Manage Passwords

## How to use
1. Copy the repo
2. Run in Python 3.8+
3. Register an account
4. Enjoy your time
